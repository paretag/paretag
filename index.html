In this tutorial you will learn steps to install php in your android mobile
First of all Download termux from f-droid or use below link
Open termux and type below command then press enter
apt update && apt upgrade
press y when it asks to continue

Type below command and enter
pkg install php
press y when it asks to continue and n when n is default

Give necessary permission to read files by writing below command
termux-setup-storage
Allow to read and write files when prompt

Make a folder with htdocs name in internal storage manually by any file manager app
Go to htdocs folder by typing below command and enter
cd /sdcard/htdocs

Start php server by typing below command
php -S localhost:8080 -t /sdcard/htdocs

Open chrome and type localhost:8080 and hit enter in address bar here you can see your all php,html page running which you will in htdocs folder. Now You just need a code editor or a file manager made by php itself by which you can make php file
